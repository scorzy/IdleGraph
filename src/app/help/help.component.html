HOW TO PLAY

Idle Graph is an Idle Incremental. The main goal is to reach enough main currency to prestige. When prestige you will star a new semi random Graph and you get some skill points.

Graph In this game there is a tree of resource generators. The root node is your main currency, every node has a level, that is the distance from Main, sarting from 1. A -> b implies that A make B Main node, leafs and other node have different colors. Buy With the Buy button you can buy more node, by buying node you also get a "bought bonus" that increase productivity. Producer With the Producer button you can buy a producer of the current node. Ticks Nodes production is based on /tick, not os "per second". You can increase your tick per second to maximize production. Sacrifice You can sacrifice node of level greater than 2 to gain a production bonus. Sacrifice will decrease the quantity of all node produced by this to zero, recursive. Sacrifice bonus is based on level and quantity of sacrificed nodes and level of this current node. New sacrifice bonus will substitute your previous sacrifice bonus, you can only sacrifice if the new sacrifice bonus is greater of actual bonus. Max Node The maximum number of node you can have is capped. It start from 50 but there are skills to increase it. Collapse Node of level greater than 2 can be collapsed. Collapse will group all producer of the current node by level, quantity sacrifice bonus and bought quantity will be the sum of collapsed nodes.
Collapse will decrease the quantity of all node produced by this to zero, recursive. With collapse you will be able to buy new node of higher level. Remove Remove button will remove the current node and all producers. You will not get any bonus. Use this when you need more node of higher level. Soft Reset This is the first level of prestige. You will restart with only the main node but with a small tick speed boost and a sacrifice boost. Prestige This is the actual prestige system. It reset everything except skills and time. Prestige gives you skill points and you can get more prestige points per prestige. Earned skill points are based on lifetime earned skills point. With prestige you will need to select a new Graph with some random bonus, this bonus substitute the previous bonus. Time Some skills will give you "Time". Time can be spent to speed up the game. To sped time click on the clock in the top right. Auto Buyers Auto Buyers will automate the purchase of some stuff. All autobuyers are available from the start but you need some skill to increase the maximun number of autobyer you can use at the same time. Priority determinate which autobuyer buy first if more autobuyers can buy in the same tick. Lower number will result in more priority.
